Plugin que habilita um widget para integração de um formulário de subscription ao RDStation.

== Description ==

Este plugin gera um formulário de subscription ideal para ser utilizado no sidebar do seu blog wordpress.

== Installation ==

O jeito mais simples de instalar é pelo seu painel do Wordpress ou faça o download e instale o arquivo ZIP em seu site.
